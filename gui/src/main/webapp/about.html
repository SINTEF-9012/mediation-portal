About Us

The SINTEF mediation has been entirely designed and developped in SINTEF ICT since 2010. It is one of the output of a more general activities addressing service interoperability using MDE techniques.

This mediation portal is:

  • mostly developed in Scala for the services, using HTML/javascript for this GUI
  • documented
  • built and deployed using maven
  • tested using JUnit, Spec2, and Jenkins
  • available on github as open-source under LGPL license

Should you have any comments or suggestion, drop us an email at mediation@sintef.no
